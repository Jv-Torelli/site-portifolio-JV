Sobre mim

// joao-victor-torelli.json

Sou João Victor Torelli de Matos, desenvolvedor formado como Técnico em Desenvolvimento de Sistemas pela Etec de Guaianases. Minha jornada no desenvolvimento é marcada por projetos que vão além do código — são soluções com beneficiários reais e impacto concreto.

No dia 05/08/2025 iniciei minha jornada no Bradesco, atuando no mercado financeiro na vertical de Canais Digitais e contribuindo para o desenvolvimento de uma plataforma interna corporativa voltada à evolução contínua dos serviços digitais da organização.

Tenho experiência na construção e evolução de aplicações backend e APIs, utilizando principalmente Java 17+ e Spring Boot. Também trabalho com bancos de dados como PostgreSQL, MySQL e MongoDB, além de arquiteturas baseadas em microserviços, containers com Docker e integração com plataformas cloud como AWS e Azure.

Atualmente estou desenvolvendo o TurboTech, um sistema para automação de oficinas mecânicas focado na geração de orçamentos digitais e gestão operacional. Ao longo da minha trajetória também desenvolvi plataformas web, sistemas de monitoramento IoT e APIs seguras aplicando boas práticas como Clean Architecture, SOLID e testes automatizados.

Acredito que engenharia de software vai além de escrever código — trata-se de construir soluções sustentáveis que gerem valor real para as pessoas e para o negócio, sempre buscando evolução técnica e qualidade no que é entregue.

Tenho certificações Microsoft AZ-900, DP-900 e Scrum Fundamentals, o que complementa minha visão técnica com fundamentos de nuvem e gestão ágil de projetos.
